Respostas dos Exercícios HTML

Pergunta Resposta
1. O que é o HTML e qual é o seu papel na construção de páginas web? HTML é a linguagem que usamos para criar a estrutura das páginas web. Ele organiza os elementos da página, como títulos, parágrafos, imagens, e links, definindo o que vai aparecer e onde.
2. Descreva brevemente a estrutura básica de um documento HTML. A estrutura básica de um documento HTML começa com a tag <html> que envolve todo o conteúdo. Dentro dela, temos a tag <head>, onde colocamos informações sobre a página, como o título, e a tag <body>, onde todo o conteúdo visível fica.
3. Como o HTML se relaciona com outras tecnologias da web, como o CSS e o JavaScript? HTML fornece a estrutura da página, CSS é usado para estilizar essa estrutura, tornando-a visualmente atraente, e JavaScript é o que dá vida à página, tornando-a interativa e dinâmica. Juntos, eles criam uma experiência completa para o usuário.
4. Cite três tags HTML básicas e explique o propósito de cada uma delas. <h1>: Usada para criar títulos principais na página, como o cabeçalho de um artigo.
<p>: Usada para escrever parágrafos de texto, onde o conteúdo é inserido.
<a>: Usada para criar links que levam o usuário para outras páginas ou seções.
5. Dê exemplos de situações em que você as usaria ao construir uma página web. <h1>: Usaria no título da página, como "Bem-vindo ao meu site".
<p>: Usaria para escrever um texto explicando sobre o site.
<a>: Usaria para linkar uma página "Sobre Mim" ou qualquer outro conteúdo relevante.
6. Qual é a importância das tags de listagem, como <ul> e <ol>, no contexto da estruturação de conteúdo em uma página? Tags de listagem como <ul> (lista não ordenada) e <ol> (lista ordenada) são importantes para organizar informações em formato de lista. <ul> é usada quando a ordem não importa, e <ol> quando a ordem dos itens é importante.
7. Dê exemplos de quando você escolheria uma sobre a outra. Usaria <ul> para uma lista de hobbies, pois a ordem não importa. Usaria <ol> para listar os passos de uma receita, onde a ordem é crucial.
8. Como a tag <a> é utilizada para criar links em HTML? A tag <a> cria links usando o atributo href para definir o destino do link. Você pode linkar para outra página, um arquivo, ou até uma parte específica da mesma página usando âncoras.
9. Como você pode vincular para outras páginas, para arquivos locais e criar âncoras dentro de uma página? Para outras páginas, basta colocar a URL no href. Para arquivos locais, coloque o caminho do arquivo. Para âncoras, use href="#id" e adicione o id correspondente na seção da página.
10. Além do atributo href, quais outros atributos importantes a tag <a> pode ter? Outros atributos importantes são target, que define onde o link será aberto (ex.: _blank para uma nova aba), e title, que mostra uma dica quando você passa o mouse sobre o link.
11. Descreva a diferença entre os atributos src e alt em relação à tag <img>. O atributo src define o caminho para a imagem que será exibida, enquanto o atributo alt fornece um texto alternativo que descreve a imagem caso ela não carregue, importante para acessibilidade.
12. Por que é importante fornecer um atributo alt adequado para imagens? O atributo alt é importante porque ajuda pessoas com deficiência visual a entenderem o que a imagem representa, e também melhora a acessibilidade e SEO do site.
13. Quais outros tipos de mídia podem ser incorporados em uma página HTML? Além de imagens, você pode incorporar vídeos, áudios, mapas interativos, e até arquivos PDF.
14. Como você pode garantir que a mídia incorporada seja responsiva e se ajuste bem em diferentes dispositivos? Para garantir que a mídia seja responsiva, você pode usar CSS para definir tamanhos em porcentagem e utilizar media queries para ajustá-la conforme o tamanho da tela.
15. Quais são as principais tags usadas para criar tabelas em HTML? As principais tags são <table> para começar a tabela, <tr> para as linhas, <th> para os cabeçalhos das colunas, e <td> para as células onde os dados são inseridos.
16. Explique a diferença entre as tags <th> e <td>, e quando você as usaria. A tag <th> é usada para os cabeçalhos das colunas, que geralmente aparecem em negrito e centralizados, enquanto <td> é usada para os dados dentro das células. Você usa <th> para dar título às colunas e <td> para preencher a tabela com informações.
17. Embora tabelas sejam usadas para exibir dados tabulares, às vezes é recomendado evitá-las em favor de outras formas de apresentação de dados. Por quê? E quais são os cenários em que as tabelas são apropriadas? Tabelas são recomendadas apenas para dados tabulares, ou seja, informações que precisam ser organizadas em linhas e colunas, como planilhas financeiras ou horários de aulas. Elas não são recomendadas para layouts ou design de páginas, pois podem ser difíceis de manipular e tornar a página menos acessível.
18. O que são meta tags e qual é o seu papel em um documento HTML? Meta tags são usadas para fornecer informações sobre a página, como o conjunto de caracteres, a descrição para motores de busca, e palavras-chave. Elas ficam no <head> e ajudam na otimização e na acessibilidade da página.
19. Como as meta tags podem impactar a otimização de mecanismos de busca (SEO)? Meta tags, como meta descrição e meta palavras-chave, ajudam os motores de busca a entender o conteúdo da página e a indexá-la corretamente, o que pode melhorar a visibilidade da página nos resultados de busca.
20. Qual é a importância de seguir as boas práticas de codificação em HTML? Seguir boas práticas garante que o código HTML seja limpo, legível e mantenha a consistência, o que facilita a manutenção e a atualização da página, além de melhorar a acessibilidade e a compatibilidade com diferentes navegadores.
